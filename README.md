# Pedro

Este repositório contém um aplicativo Streamlit romântico criado para alguém especial.

## Como executar

1. Instale as dependências:
   ```
   pip install -r requirements.txt
   ```

2. Execute o aplicativo:
   ```
   streamlit run app.py
   ```

3. Abra o navegador no endereço indicado (geralmente http://localhost:8501).

## Sobre o aplicativo

O aplicativo é uma surpresa romântica com mensagens personalizadas, efeitos visuais e uma referência especial ao Internacional. Clique no botão para revelar as mensagens!
